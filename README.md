# FightingGameJS
## How to play

#### Player 1 (left) : W, A, D and Spacebar to attack
#### Player 2 (right) : Use arrowkeys (up, left, right) and down to attack

You can check out the game and play it here: https://cmaracujafightinggame.netlify.app
